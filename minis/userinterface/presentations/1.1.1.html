What will we be doing?


Interface Structures.


Obtaining a working knowledge of how interfaces are constructed and why they take the various forms in which we see them.

Optimizing Interface Display.


Knowing what elements are critical for the functionality of an interface, how to arrange these elements spatially, and how to eliminate uneccesary elements.

Maximizing User Engagement.


Building engagement mechanisms into the experience of the interface and functionality of the software that will keep users returning to the experience.

User Interface Design Fundamentals.


Human-Computer Interface.


The origins of the development of digital computers shaped the direction of user interface design and how we think about the dovetailing of processes.

Interfacing with Machines.


The way humans have interfaced with machines and tools for centuries can afford us some insights into how we can create effective software based interfaces.

Understanding the User.


Designing around the needs of the user and the tasks they must fulfill will help determine how the interface to the process will be formed.

History of Hardware Interfaces.


Classic Interfaces.


Early methods of interfacing with computers that are no longer in use, such as punchcards and teletypes.

Conventional Interfaces.


Conventional methods of interfacing with computer systems that we currently use, such as keyboards and mice.

Prospective Interfaces.


Prospective future methods of interfacing with computer systems, such as virtual reality headsets and gesture recognition camera systems.

History of Software Interfaces.


Software is Abstraction.


Software is the computing layer that separates the way we think about problems with how the computing hardware actually calculates the problems.

Interface is a Simulation.


Interfaces provide a type of visual metaphor for humans to understand and find relateable enough to not seem completely foreign when first encountered.

Evolving Representations.


Interface styles, purposes, and details evolve with the progression of computing hardware as well as the functional needs of the system.
