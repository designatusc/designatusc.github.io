Assistant Professor of Design

Organization of Visual Elements.


What's Important?


Consider what is most important for the user to know about and interact with in order to determine how visually prominent the elements should be.

Visual Hierarchy.


Create a visual hierarchy in order to gauge the importance of content by its visual prominence over other elements.

Flow of Content.


The order in which a user encounters material will influence their perception of the content and how well they understand the subject.

Wireframing Content.


Blocking out Space.


Wireframing allows the designer to block out the areas of the screen where elements will reside, allowing for a quick structural view of the interface.

Logical Organization.


The more logical the organization of elements in the interface, the more intuitive it will be for users to engage with them.

Layout Consistency.


During the wireframing process, the designer often decides on a consistent approach to laying out the interface across different screens.

Responsive Design.


Mobile First.


Mobile access to Internet based applications surpassed desktop computers back in 2014, so it's important to begin with the primary user demographic.

Fluid Grid.


Responsive elements rest on an invisible grid that expands and contracts based on the orientation and display size of device screens.

Progressive Enhancement.


Beginning with presenting the minimum required content and building up in complexity to accommodate devices with a greater range of functionality.

The Cursor.


Indicator for Input.


Indicates the position on screen where input from the user is expected to be received. This applies to the keyboard and mouse.

Connecting Software and Hardware.


Creates an associative link between a virtual element in the software interface and a physical instrument in the hardware interface.

Cursor Actions.


Hardware and software combined create opportunities for interaction such as clicking, hovering, and dragging and dropping.

The Touchscreen.


Accommodating Fat Fingers.


Transitioning the cursor to the touchscreen requires scaling interface elements up to accommodate for the much less precise human finger.

Finger Actions.


Building off of the cursor actions of clicking and dragging and dropping, finger actions include swiping which uses a vector with velocity.

Multitouch Gestures.


As multiple fingers can be used on a touchscreen simultaneously, this opens the opportunity for the application of multitouch gestures to provide control.
