World Wide Web.


Website Purpose.


Whether it's a portfolio, ecommerce shop, social network, or web application, the intention of the site will frame its context.

Functionality.


Functionality of the website must come through as interface elements, so functionality will determine the required parameters of the interface.

Presentation.


The style of presentation of interface elements helps frame the context of the site by changing the users perception of how they should be experiencing the content.

Computer Applications.


Utilities.


Focused on performing a task and accomplishing a goal, usually framed within standard interface conventions.

Games.


Focused on user enjoyment, the interface reflects the operations of the game and the perspective of the player.

Mobile Interface.


Locative Media.


Interaction with physical space. Location supplies the context for functionality and interactions.

Telemetry Sensors.


Mobile devices come with a number of orientation, acceleration, and gyroscopic sensors that can be used as input to control interfaces.

Haptic Feedback.


Mobile devices come equipped with audio as well as vibration actuators that can be used to alert the user with specific patterns they will begin to recognize.

Kiosks.


Informational.


Allow a user with a question to quickly and easily look up the answer. Should be specific to the context in which the user is currently in.

Utility.


Allow a user to perform a specific task and accomplish a goal as quickly and easily as possible. Should be specific to the context in which the user is currently in.

Single Purpose.


Kiosks designed around a single purpose produce faster results and a quicker turnover of users allowing more participants to accomplish their goal.

Heads-Up Displays.


Stereoscopy.


The ability to see graphic elements in stereo vision in order to achieve the sense of depth perception.

Augmented Reality.


Superimposing virtual elements on top of the real world environment in order to simulate the merging of realities.

Spatial Interface.


Using elements of the real world environment as interface, turning surfaces and objects into interactive elements.

Alternative Interfaces.


New Hardware.


New mechanisms for taking advantage of different ways that the human body can interface with machines.

New Software.


With advancements in computing power, representations of interfaces can become more detailed and more interactive.

New Paradigms.


The way we think about our access to information and our performance of operations can shift drastically to define new opportunities for interfaces.
